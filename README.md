# 📝 To-Do Web Application

## Task 4 - SkillCraft Technology Web Development Internship

This project is a responsive and interactive To-Do Web Application developed as part of Task 4 of the SkillCraft Technology Web Development Internship.

The application helps users efficiently manage their daily tasks by allowing them to add, edit, delete, organize, and track task completion status.


## 🚀 Features

✅ Add New Tasks

✅ Edit Existing Tasks

✅ Delete Tasks

✅ Mark Tasks as Completed

✅ Task Completion Status

✅ Date & Time Selection

✅ Responsive Design

✅ Interactive User Interface

✅ Local Storage Support

✅ Mobile-Friendly Layout

✅ Modern Card-Based Design

✅ Smooth Animations & Hover Effects


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API



## 📋 Application Workflow

1. Enter a task name.
2. Select task date and time.
3. Click "Add Task".
4. View tasks in the task list.
5. Mark tasks as completed when finished.
6. Edit task details anytime.
7. Delete unwanted tasks.
8. Tasks remain saved using Local Storage.
